# Alchitry Interface

[![](https://jitpack.io/v/alchitry/Alchitry-Interface.svg)](https://jitpack.io/#alchitry/Alchitry-Interface)

This is a library that was originally part of Alchitry Labs V2 but was broken out to allow it to be used by other projects.

It provides a cross-platform interface for interacting with Alchitry Labs V2 hardware in Kotlin.

To use it, add it as a dependency to your Gradle project with the following.

```kotlin
dependencies {
    implementation("com.github.alchitry:Alchitry-Interface:-SNAPSHOT")
}
```

You can also replace `-SNAPSHOT` with a specific commit or version number.
